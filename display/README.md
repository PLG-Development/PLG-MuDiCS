# PLG-MuDiCS Display

## Runtime Requirements

- Bash
- Chromium
- LibreOffice (presentation view)
- ImageMagick (file preview)
- ffmpeg (file preview)
- Ghostscript (file preview)

## Development Setup

- install Go (at least version v1.25)
- build template: `go generate ./..`
- run code: `go run .`
